
# ChefBoy
Enhancing the culinary experience, this innovative application leverages TypeScript and Firebase, and integrates with the Google Vortex API for a seamless user interface. Users upload photos of their ingredients, which the app analyzes to identify and compile into a list. This list is then sent to the Google Vortex API, which returns a variety of matching recipes.

Elevating user convenience, the app offers extensive filtering options. Users can refine their search based on cuisine type, preparation time, dietary preferences, and cooking methods. Additionally, the app features user-friendly tools for rating and saving favorite recipes, and a shopping list function to add ingredients directly from chosen recipes.

Fostering a community of food enthusiasts, the app allows users to share their own recipes and tips, encouraging culinary exploration and sharing. This community aspect not only makes cooking more interactive but also enriches the overall user experience.

In summary, this application revolutionizes the way users decide what to cook by utilizing available ingredients, providing personalized recipe suggestions, and creating a collaborative platform for cooking enthusiasts.
